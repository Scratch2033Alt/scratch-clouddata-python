ScratchWarp OpenCloud — docs (content-only)

ScratchWarp has introduced OpenCloud, a remake of Scratch's clouddata. You might be seeing this as unusual, but you are actually seeing the 404 error page! If you are a developer, please read this docs.

Limits

  • Numeric only.
  • Text limit increased to 10000 characters.
  • Proxies to Turbowarp if we don't have the project clouddata — Not great if Turbowarp clouddata is down.

How To Use it

OpenCloud clouddata is the same as Scratch's clouddata, but if you are using non-browser user agent, your log will go to /bots instead of /logs.

If you want to be logged to /logs, please use a browser-like User Agent and give us the cookies needed for authentication.
